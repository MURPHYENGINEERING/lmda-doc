With the special case of a function that\textquotesingle{}s declared in one module (the {\itshape interface} module) and defined in another (the {\itshape implementation} module)\+: {\ttfamily lmda\+\_\+interface\+\_\+function\+\_\+impl()}. Here the prefix {\ttfamily lmda} indicates the broadest scope, and the suffix {\ttfamily impl} indicates the separated declaration and definition.
