\doxysubsection{Detailed Description}
A plugin packages some functionality in a form that can be swapped out live while the program is running (\char`\"{}hot reloading\char`\"{}). 

Plugins have transient internal state, but they also have external state that persists through hot reloads. Plugins run within an App\+Contextt, which provides an interface to their private and shared state, to the underlying platform, and to other game services.
