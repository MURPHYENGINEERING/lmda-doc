Simulation time, i.\+e. 

what would be read from a simulated wristwatch. This is computed by accumulating the value of {\ttfamily Dt} (time between frames), which is the time reference for all simulation dynamics (as {\ttfamily dt}); so this value reflects only time that has been experienced by the world, and ignores the accumulating error that comes from timing jitter. If the nanoseconds are in {\ttfamily Dt} then they\textquotesingle{}re in {\ttfamily t}. \mbox{\Hypertarget{struct_sim_clock_a77badbf66cf6a239cbfeb36d8d1118d7}\label{struct_sim_clock_a77badbf66cf6a239cbfeb36d8d1118d7}}
