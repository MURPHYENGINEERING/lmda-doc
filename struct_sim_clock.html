nanos_t SimClock::Dt

Time in nanoseconds elapsed between the starts of the previous and current ticks.

This is the basis for dt, which is the basis for all world dynamics, and is accumulated in t.

◆ t

nanos_t SimClock::t

Simulation time, i.e.

what would be read from a simulated wristwatch. This is computed by accumulating the value of Dt (time between frames), which is the time reference for all simulation dynamics (as dt); so this value reflects only time that has been experienced by the world, and ignores the accumulating error that comes from timing jitter. If the nanoseconds are in Dt then they're in t.

nanos_t SimClock::tpsAccumNs
